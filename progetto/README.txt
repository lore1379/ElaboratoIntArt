La rete bayesiana in esame è asia.net visualizzabile con Hugin Educational. 

Il progetto è realizzato in python, per utilizzare il programma è necessaria la libreria pysmile
(wrapper python) che presenta metodi utili alla costruzione di una rete bayesiana e una licenza accademica
ottenibili su https://download.bayesfusion.com/files.html?category=Academia

Requisiti:
	1. Python2 (ho usato python 2.7.15)
	2. import pysmile
	3. import pysmile_license
	4. import AsiaNetwork as a
	5. run as.Asia()

La spiegazione dei metodi utilizzati è data nella relazione ed è stata fornita da BayesFusion, LLC
